# HealthyStep

It is an app of my own to count time, steps, distance in meters and calories
while training sessions e.g., walking or running, 
results can be saved and displayed as a table and charts.
The makes use of Firebase Authentication, Cloud Firestore and Charts frameworks. 
Apple frameworks are UIKit, Core Motion (CMPedometer and CMPedometerData). 
UI is made by means of Xibs.
